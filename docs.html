Cheatsheet Module Documentation

Module Overview

The Cheatsheet module provides quick references for mathematical operations, JSON handling, time management, and automated testing in Python.

Math Class (`mc`)

Methods:

  • norp(num): Returns 'P' for positive, 'N' for negative, None for zero.
  • arraycal(num, lst): Adds a number to each element in a list.
  • add(a, b): Returns the sum of two numbers.
  • sub(a, b): Returns the difference between two numbers.
  • mul(a, b): Returns the product of two numbers.
  • div(a, b): Returns the division of two numbers.

Time Class (`tc`)

Methods:

  • time(): Returns the current time.
  • date(): Returns the current date.
  • tz(): Returns information about the local time zone.
  • timer(count): Pauses execution for a specified number of seconds.

JSON Class (`jc`)

Methods:

  • extract(file): Loads and returns the contents of a JSON file.
  • take(file, key): Extracts a value from a JSON file by key.

Auto Testing Class (`auto_test`)

Method:

  • run(data, config): Executes code from a file and checks if its output matches expected results from a config file.
